# Документація по виконаному завданню: Розгортання FastAPI застосунку з Nginx та Systemd Sockets

Цей файл є єдиною точкою входу в документацію проєкту автоматизованого розгортання веб-застосунку в операційній системі Ubuntu Server.

---

## 1. Варіант індивідуального завдання
* **Студент:** групи ІМ-44
* **Спеціалізація:** Інформаційні технології / Системне адміністрування та архітектура розгортання.
* **Опис конкретного індивідуального завдання:** Розробка RESTful API веб-застосунку на базі фреймворку FastAPI та автоматизація його розгортання в ізольованому середовищі віртуальної машини. Зв'язок між фронтальним веб-сервером (Reverse Proxy) та сервером додатків (Uvicorn) реалізовано через швидкі локальні **Unix Domain Sockets** за допомогою механізму сокет-активації `systemd.socket`. Для оптимізації споживання ресурсів ВМ додаток запускається динамічно через системний файловий дескриптор (`--fd 3`) лише при надходженні запитів.

## 2. Документація по розробленому веб-застосунку

### Призначення застосунку
Застосунок виконує роль RESTful API сервісу для обліку та керування сутностями (нотатками) в реляційній базі даних. Проєкт розроблено за модульним принципом із використанням автоматичного накату міграцій структури бази даних SQLite при кожному старті сервісу.
