## 2. Документація по розробленому веб-застосунку

### Призначення застосунку
`mywebapp` — це веб-сервіс на базі фреймворку FastAPI, призначений для створення, перегляду та збереження текстових нотаток. Сервіс підтримує автоматичну міграцію структури БД при старті та розмежування форматів відповідей.

### Налаштування середовища (Development / Testing)
Для локального запуску та розробки проекту без використання системних демонів Linux необхідні:
* Python версії 3.10+
* Встановлені залежностей з файлу `requirements.txt`

Створення віртуального оточення та запуск:
\`\`\`bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 migrate.py  # Накат міграцій бази даних
uvicorn app.main:app --host 127.0.0.1 --port 8000
\`\`\`

### Документація по API-ендпоінтах
Застосунок реалізує механізм **Content Negotiation** (залежно від заголовка `Accept` віддає чистий `application/json` або просту сторінку `text/html` з даними у вигляді таблиць).

* `GET /` — Кореневий ендпоінт. Очікує та віддає тільки `text/html`. Повертає список усіх доступних ендпоінтів бізнес-логики.
* `GET /notes` — Вивести список усіх нотаток (поля `id`, `title`).
* `POST /notes` — Створити нову нотатку. Приймає `title` та `content`.
* `GET /notes/<id>` — Вивести повний вміст конкретної нотатки за її `id`.
